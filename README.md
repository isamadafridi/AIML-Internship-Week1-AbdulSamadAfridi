# Student Performance EDA Project

## 👤 Name:
Abdul Samad Afridi

## 📌 Project Title:
Exploratory Data Analysis of Student Performance Dataset

## 📊 Dataset Used:
Students Performance in Exams (Kaggle Dataset)

## 🧰 Tools & Libraries Used:
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 🔍 Key Findings:

- Students perform better in **reading and writing** compared to math.
- **Test preparation course significantly improves performance**, especially in writing (+15% improvement).
- **Parental education level has a strong positive relationship with student scores**.
